# MotoRide - Сайт по продаже мотоциклов

## Описание
Веб-сайт для продажи мотоциклов с полностью кликабельным интерфейсом, адаптивным дизайном и динамическим каталогом.

**Функциональность:**
- Просмотр каталога мотоциклов
- Карточки товаров с фото, описанием и ценой
- Кнопки "Подробнее" и "Купить" с тост-уведомлениями
- Форма обратной связи с валидацией
- Адаптивная вёрстка под все устройства
- Плавная навигация по разделам

## Технологии
- HTML5
- CSS3 (Flexbox, Grid, медиа-запросы)
- JavaScript (ES6)
